Detailed Description

template<class T, class Coordinate>
class PRISMS::Bin< T, Coordinate >

A class for binning things template class T: the type of thing being binned

  • should have T::operator==() defined to use 'Bin::add_once' template class Coordinate:
  • expected to be 'vector<double> like'
  • should have Coordinate::operator[]() defined

Construct a Bin 'min': minimum value of each coordinate component 'incr': bin spacing along each direction (this is uniform along each dimension) 'N': number of bins along each direction

For example, to bin the range (0->10, 0->20, 10->100), with size 1 bin spacing: Bin<T>( {0,0,10}, {1,1,1}, {10, 20, 90})
